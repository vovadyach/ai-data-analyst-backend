# Architecture

## Overview
```
[Next.js Frontend] → [FastAPI Backend] → [PostgreSQL]
                          ↓
                    [Google Gemini AI]
```

## Services
- **Backend**: FastAPI REST API
- **Frontend**: Next.js (separate repo)
- **Database**: PostgreSQL (Docker)
- **AI**: Google Gemini

# Development Setup

## Prerequisites
- Python 3.12+
- Docker Desktop
